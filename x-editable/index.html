X-editable

可以使用 Twitter Bootstrap, jQuery UI 或者纯 jQuery 进行编辑

该插件可以在页面上创建和编辑元素。可以和（bootstrap，jquery-ui，jquery库）配合使用，包括 弹出(popup) 与 内联(inline) 两种模式可以选择。点击Demo查看演示。

演示 (new window) 查看文档

当前版本 1.5.1 修改日志

↓ 选择与X-editable配合使用的核心库 ↓

Bootstrap (2 or 3)

用Bootstrap编辑 样式文档地址， 弹出模式 文档地址。 日期输入模式采用 bootstrap-datepicker 库。

jQuery UI (1.9+)

最低支持 jQuery UI 1.9 版本 提示插件。 结合按钮样式编辑弹出窗口 Button widget。 可以使用任何主题的 jQuery UI Themeroller。

jQuery (+ poshytip plugin)

X-editable基于jQuery，内联模式与弹出模式引入jQuery即可。 对于日期字段可以引入 combodate 或者 momentjs 库。

AngularJS 项目

Angular-xeditable

简述

  • 弹出(popup) 与 内联(inline) 模式
  • 支持input类型:
    • text
    • textarea
    • select
    • date
    • datetime
    • dateui
    • combodate
    • html5types
    • checklist
    • wysihtml5
    • typeahead
    • typeaheadjs
    • select2
  • 客户端和服务端验证
  • 容器管理
  • 点击或者双击切换
  • 支持快捷键 (escape/enter/tab)
  • 支持ie7+，和所有主流浏览器
  • 单元测试
  • 事件

什么是 editable?

  • 快读编辑form表单
  • 减少代码量
  • 增加 / 删除字段
  • 用于后台数据修改
  • 更好的用户体验
